# Insurance for Dummies: Personal Lines

A plain-language study companion for the personal lines portions of the **CPCU** and **CIC** designation programs. No terminal, no code — just read along on GitHub.

---

## Who This Is For

You're studying for your CPCU or CIC (or both) and want a clear, jargon-light reference that connects the dots between topics. This isn't a replacement for the official courseware — it's the study buddy who explains things in normal English.

---

## How to Use This Repo

Read the files in order within each section. Each file covers one major topic area and includes:

- **Key concepts** explained in plain language
- **What you need to know** for the exam
- **How it connects** to other topics
- **Practice questions** to test yourself

## Program Comparison at a Glance

| | CPCU | CIC |
|---|------|-----|
| **Administered by** | The Institutes | Risk & Insurance Education Alliance |
| **Total courses** | 8 + ethics | 7 institutes (pass 5) |
| **Personal lines portion** | 2 courses (555, 556) | 1 institute (16 hours) |
| **Exam format** | 50 MCQ, 65 min, virtual | Exam after 16-hr institute |
| **Focus** | Deeper technical + strategic | Practical coverage knowledge |
| **Best for** | Underwriters, risk managers, claims | Agents, brokers, account managers |
